# AutoHeal – Self-Healing UI Automation Framework

AutoHeal is a Selenium-based UI automation framework designed to reduce test failures caused by frequent UI changes.

## Key Features
- Self-healing locators with fallback strategies
- DOM-based heuristic recovery
- Retry logic for flaky tests
- Parallel execution using TestNG
- Jenkins CI-ready

## Tech Stack
Java, Selenium WebDriver, TestNG, Maven, Jenkins

## Use Case
Automatically recovers broken locators at runtime, reducing test maintenance and improving execution stability.
